API Endpoints

The following API endpoints are available for testing:

  • /api/minimal - Ultra-simple implementation with minimal dependencies
  • /api/frame-action - More advanced handler that generates dynamic SVGs
  • /api/all-routes - Combined handler that can process different request types
  • /api/simple-frame - NEW Simple frame handler with latest format
  • /api/debug - NEW Debug endpoint that shows detailed request information
  • /api/test - NEW Ultra-minimal test endpoint
  • /api/minimal-frame - NEW Absolute minimal frame handler

Troubleshooting

If you're experiencing issues with button functionality in Warpcast frames, try the following:

  • Start with the simpler implementations (super-minimal.html, static.html)
  • Make sure your API endpoints are returning valid FC frame headers
  • Check that your post_url is correctly set to your domain
  • Try using post_redirect actions instead of server-side interactions

View the GitHub repository for more details.
