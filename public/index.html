Warplet Traders

Frame URLs

Access the Warpcast Frame at any of these URLs:

  • /tiny.html (Ultra-Minimal HTML - TRY THIS ONE!)
  • /minimal.html (Very Simple HTML)

For testing in Warpcast, use: https://warplet-traders.vercel.app/tiny.html

How to Use

Share the following URL in Warpcast to interact with the mini-app:

https://warplet-traders.vercel.app/tiny.html

This frame tracks which of the people the user is following are top traders on BASE using their warplet (Warpcast's built-in wallet).

Features

  • View the top 5 most active traders among the accounts you follow
  • Toggle between 24-hour and 7-day timeframes
  • Share results via the cast composer
