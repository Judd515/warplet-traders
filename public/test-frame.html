Warplet Traders - Local Test Frame

Testing Instructions

This page includes the necessary meta tags to be recognized as a Warpcast Frame when accessed in a Frame validator.

To test in the Warpcast Frame validator:

  1. Deploy this page to a publicly accessible URL (localhost won't work for validation)
  2. Copy the deployed URL of this page
  3. Visit https://warpcast.com/~/developers/frames
  4. Paste the URL in the validator input field
  5. Test the frame interactions

Tech Stack

This project is built with:

  • Express.js backend
  • Dune Analytics API for trading data
  • Neynar API for Farcaster social graph
  • Dynamic SVG generation

Note: Local testing with localhost URLs won't work in the actual Warpcast validator, as it needs to access your frame publicly.
