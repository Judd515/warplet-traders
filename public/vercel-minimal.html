Ultra-minimal version for Vercel Hobby Plan

Important Notes for Vercel Deployment

This is an ultra-minimal version designed specifically for Vercel's Hobby Plan:

  • Uses a single /api/index.js file for all frame interactions
  • Images are hosted on Imgur instead of using base64-encoded SVGs
  • Simplified HTML with minimal meta tags
  • No complex routing or dependencies

When deploying to Vercel:

  1. Make sure to deploy the entire repository
  2. Set Framework Preset to "Other"
  3. The /api folder will be detected automatically
  4. No special build steps are required
