Frame Debug Tool

This page helps diagnose frame communication issues.
When testing in a browser, click the buttons below to simulate frame interactions.

How to use this debug tool

  1. When deployed to Vercel, access this page at https://warplet-traders.vercel.app/debug-frame.html
  2. Click any button to send a test request to the /api/frame-debug endpoint
  3. Check the server logs in Vercel to see detailed request information
  4. This helps identify issues with headers, CORS, or payload format
